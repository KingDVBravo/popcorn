# Game in C (idk why i named it popcorn)
Game made using C and SDL2

## Run Locally

Clone the project

```bash
  git clone https://github.com/KingDVBravo/popcorn
```

Go to the project directory and build directory

```bash
  cd popcorn
  mkdir build
  cd build
```

Build and run using CMake

```bash
    cmake ..
    make
    ./popcorn
```
